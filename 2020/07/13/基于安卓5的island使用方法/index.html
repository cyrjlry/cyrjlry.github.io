教程开始

  1. pad恢复出厂设置，先不要下载任何非此教程的第三方应用。

  2. 下载island，正常情况下只会卡在引导屏，无法进入应用。

  3. 在pad设置里打开开发者选项（具体路径：设置→关于手机→连点五次版本号）

  4. 开发者选项里打开usb调试。

  5. 麻烦：pad连接电脑键入cmd命令（详情见文章末尾）

  6. 简单：利用秋之盒一键激活island。（具体连接方法跟手机连接电脑一样）

  7. pad重启island应用，左滑进入壶中界设置，打开adb安全结界（此功能一旦打开，开发者选项所有功能将无法使用，如需关闭，请在island设置里撤销island的管理权限，否则一定会有不良后果）

  8. enjoy it！

    PC端cmd激活注意事项

  • 初始化 Demigod mode 的 ADB 指令：（须先移除全部账号，但无需销毁 Island 空间）dpm set-profile-owner --user 0 --name Mainland com.oasisfeng.island/.IslandDeviceAdminReceiver

  • 激活island：adb -d shell dpm set-device-owner com.oasisfeng.island/.IslandDeviceAdminReceiver

  • 大部分设备上，至此就大功告成了！现在运行 Island 就可以开始体验了。

  • 如果最后的 ADB 指令不成功，则可能还需要执行以下额外的指令：

  • adb -d shell settings put global device_provisioned 0

  • adb -d shell dpm set-device-owner com.oasisfeng.island/.IslandDeviceAdminReceiver

  • adb -d shell settings put global device_provisioned 1

  • 此种方法极其不稳定，并且有的电脑还需要手动安装驱动，所以不推荐。
